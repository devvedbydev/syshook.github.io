SystemHook Documentation

Getting Started:

  1. !redeem <your key> in commands
  2. Uninstall anti-virus (adding an exception won't work)
  3. !download in commands, download it and extract to desktop
  4. Run hwid.exe as admin and copy HWID
  5. !register <your hwid> in commands
  6. Run driver.exe as admin
  7. Visit http://localhost:8000/ to access the code editor

Custom Functions

sys_post

The sys_post function allows you to send HTTP POST requests.

local response = sys_post({
    Url = "https://discord.com/api/webhooks/123123123/blablabla",
    Body = "content=onion"
})
print(response.StatusCode)
